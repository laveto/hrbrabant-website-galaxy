# Laveto's 🚀 Galaxy
De standaard Laravel installatie, waar we continue op voortborduren.

## USP (Unique Selling Proposition)

- Een admin template
- CRUD tools
- Basic user management
- Website implementatie met standaard modules
- Handige helpers
- Een structuur waar jij en je collega's bekend mee gaan zijn


## Galaxy principles
We streven naar het toepassen van de volgende principles:<br />

- 🚀&nbsp;&nbsp;Gebruik voor élk Laravel project Galaxy, voor een snelle take-off.<br />
- 👍&nbsp;&nbsp;Geef gemaakte en interessante oplossingen terug aan Galaxy.<br />
- 🤔&nbsp;&nbsp;Typ je vaak hetzelfde, denk dan er over na om het generiek te maken en te doneren.<br />
- 💰&nbsp;&nbsp;Streef naar een Minimum Value Product en maak geen onnodige complexe oplossingen.<br />
- 💪&nbsp;&nbsp;Verbeter Galaxy als team.<br />
- ❤️️&nbsp;&nbsp;Maak met liefde.<br />
- 🔁&nbsp;&nbsp;Leer deze principles uit je hoofd
